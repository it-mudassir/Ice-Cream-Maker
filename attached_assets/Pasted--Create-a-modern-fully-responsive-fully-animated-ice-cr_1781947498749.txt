

Create a **modern, fully responsive, fully animated ice cream parlour website** with a fun, vibrant, and premium UI/UX design. The website should feel playful yet high-end, with smooth animations, micro-interactions, and engaging visuals.

---

### 🎨 **Design Style & Theme**

* Use a **bright, pastel-based color palette** (soft pinks, mint green, creamy beige, lavender, sky blue)
* Add **glassmorphism + soft shadows + gradients**
* Use **rounded UI elements** to give a soft dessert-like feel
* Typography should be **fun, bold for headings + clean for body**
* Overall vibe: **premium dessert brand meets playful animation**

---

### 🏠 **Hero Section (Highly Animated)**

* Fullscreen hero with:

  * A **3D or illustrated ice cream cone/cup**
  * Scoops **dropping from above and stacking smoothly**
  * Melting animation subtly looping
* Background:

  * Floating elements (sprinkles, fruits, chocolate chips)
* CTA buttons:

  * “Order Now” and “Explore Flavors”
  * Buttons have hover animations (bounce / liquid fill effect)
* Smooth entrance animations (fade + slide + scale)

---

### 🍨 **Menu / Flavors Section**

* Grid or carousel layout with:

  * Ice cream cards (each flavor)
* Each card includes:

  * Image
  * Name
  * Price
* Animations:

  * Hover → scoop slightly rotates or lifts
  * Add-to-cart button appears with animation
* Filter system:

  * Categories (Chocolate, Fruit, Special, Vegan)
  * Smooth animated filtering (no page reload)

---

### 🧁 **Featured Specials Section**

* Highlight premium or seasonal flavors
* Use:

  * Horizontal scroll with parallax effect
* Each item:

  * Has glowing or animated border
  * Subtle floating animation

---

### 🛒 **Interactive Ordering System (Frontend Only)**

* Add-to-cart animation:

  * Item “flies” into cart icon
* Cart drawer:

  * Slides from right
  * Smooth item add/remove animations
* Quantity selector with animated counters

---

### 📖 **About Section**

* Story of the brand
* Include:

  * Animated illustrations or SVGs
  * Scroll-triggered animations (fade, reveal, slide)

---

### 📍 **Store Locations / Contact**

* Interactive map section
* Cards for locations with:

  * Hover animations
* Contact form:

  * Floating labels
  * Smooth validation animations

---

### ⭐ **Testimonials Section**

* Carousel of customer reviews
* Smooth sliding animation
* Avatar images with soft glowing borders

---

### 📸 **Gallery Section**

* Masonry layout
* Lightbox popup with smooth zoom animation
* Hover → image zoom + overlay effect

---

### 🎉 **Extra Animation Details (IMPORTANT)**

* Use:

  * Scroll-triggered animations (GSAP / Framer Motion style)
  * Parallax effects
  * Micro-interactions on all buttons and cards
* Add:

  * Floating blobs / shapes in background
  * Cursor interaction (optional: trailing sprinkles or glow)
* Page transitions:

  * Smooth fade or slide between pages

---

### 📱 **Responsiveness**

* Fully responsive across:

  * Desktop
  * Tablet
  * Mobile
* Animations should adapt smoothly to all screen sizes

---

### ⚙️ **Technical Expectations**

* Clean, modular structure
* Use:

  * HTML, CSS, JavaScript (or React)
* Include:

  * Reusable components
  * Smooth performance (no laggy animations)

---

### ✨ **Overall Goal**

Create a website that feels:

* **Delicious**
* **Interactive**
* **Visually addictive**
* **Highly animated but smooth and professional**
